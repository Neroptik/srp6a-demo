Welcome to the Thinbus Secure Remote Password SRP6a Authentication with Spring Security demo!

All this application lets you do is sign-up (or change your password), logout, and login. That's all you need to integrate Thinbus SRP into your application to provide zero knowledge proof-of-password authentication!

There is also a link above to the Thinbus Configuration Converter which you can use to convert an 'openssl dhparam -text' safe prime into Thinbus configuration values.
